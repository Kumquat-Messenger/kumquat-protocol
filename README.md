# Kumquat- The Decentralised Communication Protocol

## What is Kumquat?
Kumquat is a decentralised protocol, where users use a client to send and recieve messages and content through a relay service. 

## (Planned) Features
- User Authentication via public / private keys
- End to End Encrypted direct and group messages
- Video and Audio calls via WebRTC
- Compatibility with C#, Python, and Flutter.
- Multi-Device Supporrt
